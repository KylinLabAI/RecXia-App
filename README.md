# RecXia

[中文介绍](README.zh.md)

![RecXia Logo](https://raw.githubusercontent.com/KylinLabAI/RecXia-App/master/resources/logo_256.png)

AI-powered mock interview recording management system for training institutions.

## Features

- **Multi-role system**: Administrator, Academic Affairs, Teacher, Student, and Parent
- **Cross-platform recording**: Desktop app, Android app, and Web management
- **AI analysis pipeline**: Automatic speech-to-text and intelligent evaluation
- **Video annotation**: Timeline annotations with public or internal visibility
- **Parent mini-programs**: WeChat and DingTalk apps for parents to view feedback
- **Batch management**: Excel import for users, classes, students, and parents
- **Flexible storage**: Local disk, MinIO, or Alibaba Cloud OSS
