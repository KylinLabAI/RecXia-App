# 录匣 Recxia

[English](README.md)

![录匣 Logo](https://raw.githubusercontent.com/KylinLabAI/RecXia-App/master/resources/logo_256.png)

面向培训机构的 AI 驱动模拟面试录制管理系统。

## 功能特性

- **多角色系统**：管理员、教务、教师、学员、家长
- **跨平台录制**：桌面应用、Android App、Web 管理端
- **AI 分析流水线**：自动语音转文本与智能评测
- **视频批注**：时间轴批注，支持公开或内部可见
- **家长小程序**：微信与钉钉小程序查看反馈
- **批量管理**：Excel 导入用户、班级、学员、家长
- **灵活存储**：本地磁盘、MinIO、阿里云 OSS
