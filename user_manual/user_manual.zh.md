## 简介

欢迎使用录匣。本手册按您要完成的任务分类，引导您使用每一个功能。

## 管理员指南

### 入门

#### 任务：系统设置

**前提条件：**
- 服务器环境已准备
- 存储后端已配置（本地磁盘、MinIO 或阿里云 OSS）

**操作步骤：**

1. 登录录匣 Web 管理后台
   - 您将看到：系统概览仪表盘

2. 进入系统设置
   - 配置存储路径和凭据
   - 配置 AI 服务参数（whisper.cpp 模型路径、Qwen API Key）
   - 配置微信/钉钉通知渠道

3. 保存设置并验证连接
   - 您将看到：所有服务显示绿色状态指示

**预期结果：** 所有系统服务显示健康状态。

**常见问题：**

| 问题 | 解决方案 |
|------|----------|
| 存储连接失败 | 检查凭据和网络访问 |
| AI 服务不可达 | 验证 API 密钥和模型路径 |

### 日常操作

#### 任务：用户管理

**前提条件：**
- 已分配管理员角色

**操作步骤：**

1. 进入用户管理页面
   - 您将看到：现有用户列表

2. 逐个添加用户或使用批量导入
   - 下载 Excel 模板
   - 填写用户数据（教师、教务人员）
   - 上传文件

3. 为用户分配角色
   - 您将看到：更新后的用户列表及角色

**预期结果：** 用户可以使用分配的角色登录。

**常见问题：**

| 问题 | 解决方案 |
|------|----------|
| 导入失败 | 检查 Excel 格式是否与模板一致 |
| 用户无法登录 | 验证用户名和角色分配 |

### 故障排除

#### 任务：系统维护

**操作步骤：**

1. 监控系统状态仪表盘
2. 检查存储使用情况，必要时清理
3. 查看通知发送日志

## 教务指南

### 入门

#### 任务：学员报名

**前提条件：**
- 已分配教务角色
- 班级结构已创建

**操作步骤：**

1. 进入学员管理页面
   - 您将看到：学员列表及报名状态

2. 逐个添加学员或通过批量导入
   - 填写学员姓名、家长联系方式、目标班级

3. 标记缴费状态
   - 您将看到：缴费状态已更新

4. 将学员分配到班级
   - 您将看到：班级名单已更新

**预期结果：** 学员已报名并分配到班级。

**常见问题：**

| 问题 | 解决方案 |
|------|----------|
| 重复学员条目 | 在管理页面合并或删除重复项 |
| 班级分配错误 | 通过编辑功能重新分配 |

## 教师指南

### 入门

#### 任务：启动录制场次

**前提条件：**
- 已分配教师角色
- 班级和学员已配置
- 已安装录制应用（桌面端或 Android）

**操作步骤：**

1. 登录 Web 管理门户
   - 您将看到：教师仪表盘

2. 点击「启动录制应用」打开原生录制器
   - 或直接使用 Android App

3. 创建新的模拟面试场次
   - 选择班级
   - 如需调整学员顺序
   - 确认创建

4. 学员准备好后开始录制
   - 您将看到：带计时器和控件的录制界面

5. 停止录制并上传视频
   - 您将看到：上传进度和确认信息

**预期结果：** 视频已上传，可在视频管理中查看。

**常见问题：**

| 问题 | 解决方案 |
|------|----------|
| 录制应用无法启动 | 检查浏览器的协议处理程序权限 |
| 上传失败 | 检查网络和存储配置 |
| Android App 崩溃 | 更新到最新版本 |

### 日常操作

#### 任务：视频管理

**前提条件：**
- 有可用的录制视频

**操作步骤：**

1. 进入视频管理页面
   - 您将看到：所有录制视频及筛选条件

2. 选择视频进行回看
   - 播放带时间轴的视频

3. 在特定时间点添加批注
   - 输入批注文本
   - 设置可见性：公开（家长可见）或内部（仅教师可见）

4. 查看 AI 生成的转录文本和批注
   - 批准、修改或拒绝 AI 建议

**预期结果：** 视频具有准确的教师和 AI 批注。

**常见问题：**

| 问题 | 解决方案 |
|------|----------|
| AI 转录不准确 | 手动编辑或使用不同设置重新处理 |
| 批注无法保存 | 刷新页面后重试 |

#### 任务：向家长分发视频

**前提条件：**
- 视频已审核并批注
- 学员已关联家长账号

**操作步骤：**

1. 打开视频详情页
2. 点击「分发」按钮
   - 系统生成带签名的访问链接
   - 通过微信/钉钉向家长发送通知

3. 确认分发
   - 您将看到：分发状态已更新

**预期结果：** 家长收到通知并可访问视频。

**常见问题：**

| 问题 | 解决方案 |
|------|----------|
| 家长未收到通知 | 检查通知渠道设置 |
| 链接已过期 | 重新分发以生成新链接 |

## 家长指南

### 入门

#### 任务：注册并访问视频

**前提条件：**
- 学员已由机构报名
- 已安装微信或钉钉（用于小程序）

**操作步骤：**

1. 接收机构发送的注册链接
   - 或打开微信/钉钉小程序

2. 填写家长信息
   - 家长姓名、学员姓名、联系方式
   - 提交注册

3. 等待机构审批
   - 您将看到：注册成功提示

4. 收到通知后访问视频列表
   - 查看学员的所有模拟面试视频

**预期结果：** 家长可以查看视频、转录文本和批注。

**常见问题：**

| 问题 | 解决方案 |
|------|----------|
| 注册待审批 | 联系机构进行审批 |
| 无法打开链接 | 使用支持的浏览器或小程序 |

### 日常操作

#### 任务：查看视频和反馈

**操作步骤：**

1. 从列表中打开视频
   - 视频播放器加载并显示转录面板

2. 播放视频并查看 AI 转录文本
   - 点击转录行可跳转到对应时间点

3. 查看教师和 AI 批注
   - 点击批注时间点可跳转到相关时刻

4. 如需下载视频
   - 使用下载按钮（如机构已启用）

**预期结果：** 全面了解学员的表现和反馈。

## 通用任务

### 登录

**操作步骤：**

1. 打开录匣门户或应用
2. 输入用户名和密码
3. 点击登录

**常见问题：**

| 问题 | 解决方案 |
|------|----------|
| 忘记密码 | 联系管理员重置 |
| 账号被锁定 | 等待 30 分钟或联系管理员 |

### 设置

**操作步骤：**

1. 进入个人资料/设置页面
2. 更新个人信息
3. 如需更改密码
4. 保存更改
